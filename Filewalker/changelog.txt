-- 1.3
-------------------------------------------------
* Multiple files can now be deleted at once.
* Filewalker will now remember up to 10 previously opened directories. 
  These can be selected from the "File" menu. 
* Users can now select all the files displayed in the list view
  via CTRL+A or by clicking "Select All" under the "Edit" menu.

-------------------------------------------------
* When refreshing or selecting a new directory, the listview won't remove
  its contents until the pogram has finished scanning the specified directory.
* Multiple files can now be copied at once.
* File enumeration is now performed on a seperate thread.
* File copying is performed on a separate thread.
